# TP RIC - MLP

## Description

Ce projet consiste à implémenter un réseau de neurones (MLP) en Python avec NumPy.

## Étapes réalisées

* Semaine 1 : Génération du dataset + visualisation 3D
* Semaine 2 : Implémentation du forward (MLP)
* Semaine 3-4 : Backpropagation (calcul des gradients) 
* Semaine 5 : Entraînement + visualisation des résultats
